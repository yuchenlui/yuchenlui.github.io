编程环境搭建教程：R、RStudio、Rtools、Mamba、VSCode 及相关配置
一、安装 R
访问 R 官网（https://www.r-project.org/），在首页找到 “Download” 选项进入下载页面，根据操作系统选择合适版本（如 Windows 系统点击 “Download R for Windows” 后选择 “base” 链接下载基础安装包）。
找到下载的安装包（.exe 文件）双击运行，在安装向导中依次选择安装语言（一般为英语）、安装目录（建议默认）、组件（一般默认）、启动选项（默认），最后点击 “Install” 开始安装并等待完成。
[此处插入 R 官网首页、下载页面、安装向导部分页面截图，分别标注为图 1、图 2、图 3]
二、安装 RStudio
前往 RStudio 官网（https://posit.co/），点击首页 “Download” 按钮进入下载页面，依据操作系统选择相应版本（Windows 系统选择 “RStudio Desktop” 的 Windows 版本）。
运行下载的 RStudio 安装包（.exe 文件），在安装向导中点击 “Next”，选择安装目录（可默认），按需选择创建桌面快捷方式等选项后继续点击 “Next”，最后点击 “Install” 等待安装结束。
[此处插入 RStudio 官网首页、下载页面、安装向导部分页面截图，分别标注为图 4、图 5、图 6]
三、安装 Rtools（仅 Windows 系统适用）
打开 Rtools 官网（https://cran.r-project.org/bin/windows/Rtools/），根据 R 版本选择合适的 Rtools 版本并下载。
双击运行下载的 Rtools 安装包（.exe 文件），在安装向导中点击 “Next”，选择安装目录（建议默认）、组件（默认），安装过程中选择 “Add Rtools to the system PATH for the current user” 添加系统环境变量，点击 “OK” 后继续点击 “Install” 等待安装完成。
[此处插入 Rtools 官网页面、下载链接位置、安装向导部分页面截图，分别标注为图 7、图 8、图 9]
四、安装 Mamba
在浏览器访问 Mambaforge 下载页面（https://github.com/conda-forge/miniforge#mambaforge），根据操作系统选择合适版本（如 Windows 64 位系统选择 “Mambaforge-Windows-x86_64.exe”）下载。
运行下载的安装包，在安装向导中点击 “Next”，选择安装目录（注意路径不含中文），选择 “Add Mamba to my PATH environment variable” 添加到系统路径，点击 “Next” 后点击 “Install” 等待安装完成。
[此处插入 Mambaforge 下载页面、安装向导部分页面截图，分别标注为图 10、图 11]
五、安装 VSCode
进入 VSCode 官网（https://code.visualstudio.com/），点击首页 “Download” 按钮进入下载页面，选择对应操作系统版本（Windows 系统点击 “Windows” 下载按钮）。
找到下载的 VSCode 安装包（.exe 文件）双击运行，在安装向导中点击 “Next”，选择安装目录（可默认），按需选择创建桌面快捷方式等选项后点击 “Next”，最后点击 “Install” 等待安装完成。
[此处插入 VSCode 官网首页、下载页面、安装向导部分页面截图，分别标注为图 12、图 13、图 14]
六、在 VSCode 中配置 R
打开 VSCode，点击左侧 “Extensions” 图标打开扩展面板，搜索 “R” 并安装 “R for Visual Studio Code” 扩展插件，安装完成后可能需重启 VSCode。
打开 VSCode 设置（点击左下角齿轮图标选择 “Settings”），搜索 “r.path”，若 R 安装路径未自动检测到，点击 “Edit in settings.json” 添加或修改 “r.rpath” 为 R 安装目录（如：”r.rpath”: “C:\Program Files\R\R-4.2.2\bin\R.exe”，注意反斜杠转义），保存后 VSCode 即可找到 R 执行路径。
[此处插入扩展面板搜索 R 插件、设置 R 路径相关截图，分别标注为图 15、图 17]
七、在 VSCode 中配置 Python（使用 Mamba 管理的环境）
打开命令提示符（Windows 系统搜索 “cmd” 打开），输入 “mamba create -n myenv python=3.9”（创建名为 “myenv” 的 Python 3.9 环境，可按需更改）创建 Python 虚拟环境，等待 Mamba 下载安装完成。
打开 VSCode，点击左下角 Python 版本号（若未显示可在命令面板输入 “Python: Select Interpreter” 打开菜单），选择 “Enter interpreter path…”，找到创建的虚拟环境中的 Python 解释器路径（如：“C:\Users\YourUsername\mambaforge\envs\myenv\python.exe”）点击 “OK”，VSCode 将使用该解释器运行 Python 代码。
[此处插入创建虚拟环境命令执行过程、选择 Python 解释器相关截图，分别标注为图 18、图 19、图 20]
完成上述步骤后，即可使用这些工具进行编程与数据分析工作。
